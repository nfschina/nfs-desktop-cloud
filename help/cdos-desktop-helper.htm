1 引言

1.1 编写目的

本文档是中科方德桌面云操作手册，对系统功能和操作使用的细化描述，旨在让用户更清晰明确了解桌面云的操作使用方法。

本文档面向软件项目使用人员及相关工作人员。

1.2 开发背景

本软件为中科方德桌面云客户端，由软件所服务器OS部桌面云小组开发完成。本软件系统基于中科方德云操作系统，是一套基于openstack开源架构，提供各种主流桌面操作系统的虚拟化和远程桌面连接和传输，方便用户远程使用和资源集中管理。

2 功能介绍

   方德云客户端基于中科方德云操作系统，按照用户角色权限不同，实现对虚拟机的查看、管理操作，为用户提供流畅的各类主流云桌面办公和开发体验。。

2.1 安装部署

方德桌面云提供了全系列操作系统的客户端安装包。

1）debian系linux操作系统的安装：

    打开一个terminal，运行命令：

    sudo dpkg -i cdos-desktop-cloud_0.1-10_amd64.deb

    如果提示缺少依赖的错误，运行命令：

    sudo apt-get install -f

2）redhat系linux操作系统的安装：

sudo yum install cdos-desktop-cloud_0.1-10_amd64.rpm

3）windows操作系统的安装：

双击“方德桌面云.exe”进行安装，完成后会产生相应的桌面快捷方式。

2.2 桌面云启动

debian系和redhat系linux操作系统启动方式类似：

1）系统图标启动：安装成功后，点击开始菜单―》系统工具---》云桌面客户端  打开。桌面云图标如下：

2）命令行启动：打开一个terminal，运行命令cdos-desktop-cloud

windows操作系统：

直接双击快捷方式启动，或在开始菜单找到“方德桌面云”进行启动。

2.3 桌面云登录

   点击方德云客户端快捷菜单，打开方德云客户端登录页面，输入服务器地址、用户名、密码点击登录按钮，即可成功登录系统。

图1 登录页面

 

2.4 虚拟机管理

    登录系统之后，可以查看到登录用户下的虚拟机简略图，页面如下所示：

图2 方德云首面

 

右键点击某个虚拟机会弹出开/关机、查看详细信息操作。如果当前虚拟机的状态为"RUNNING"可进行关机操作；如果虚拟机当前的状态为"SHUTDOWN",可进行开机操作。

图3 虚拟桌面开关机

    右键选择查看详细信息，可打开虚拟机的详细信息弹出框，查看虚拟机的详细信息。

图4 虚拟机详细信息页面

 

    单击虚拟机缩略图会打开虚拟机全屏显示页面，使用远程的虚拟桌面。

   页面虚拟机缩略图默认按照“视图”的方式进行展示，点击右上角的“列表”展示，可切换虚拟机的展示方式。虚拟机的列表展示页面如下：

图5 虚拟机列表展示页面

 

   在虚拟机列表中双击即可打开虚拟机全屏使用页面。

2.5 USB重定向功能

单击虚拟机缩略图会打开虚拟机全屏显示页面，使用远程的虚拟桌面。虚拟桌面和瘦客户端物理机界面之间的鼠标焦点可用Ctrl+Alt组合健切换，从虚拟桌面中切换出鼠标焦点，点击“文件――>USB设备选择”即可进行相应USB设备的选择和重定向使用。

图6 方德云USB重定向

注：目前linux桌面云客户端支持该功能。USB设备需要提前插入瘦客户机终端，并确保瘦客户机物理机系统已经识别了该系统。

2.6 其他设置

2.6.1 刷新设备

    点击右上角的的“刷新设备”按钮，可对方德云客户端展示的虚拟机所有信息进行刷新，展示虚拟机的最近的状态信息，包括虚机的开关机状态和详细信息等。

 

图7 刷新设备

 

注：右键开机或者关闭远程虚拟机后，需要等待片刻在进行设备刷新即可更新状态。

2.6.2 升级检测

    点击右上角的“升级检测”按钮，可检测当前系统是否有最近版本，可对版本选择是否升级。

图8 升级检测

 

2.6.3 退出

     点击右上角的“退出”按钮，可退出程序。

图9 退出
